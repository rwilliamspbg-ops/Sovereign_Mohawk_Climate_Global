# BETA_RELEASE

## Scope

This beta validates domain adaptation from ecosystem templates to climate-specific federated learning workflows.

## Included

- Static geospatial climate dashboard simulation.
- Policy-gated federated learning workflow placeholders.
- Initial climate compliance evidence mapping.
- Baseline operator and governance documentation.

## Exit Criteria for GA Candidate

- Live telemetry integration for climate and compliance metrics.
- Region-policy enforcement tests for cross-border climate model flow.
- Security wrapper integration tests for signed update verification.
- Reproducible artifact capture and CI quality gates.
